# rp·mexico — App Android

Repositorio de **distribución**. Aquí solo vive el APK para instalar en tablets y
teléfonos Android. El código fuente está en repositorios privados aparte.

## Descargar e instalar

1. Entra a **[Releases](../../releases)** y baja el `.apk` de la versión más reciente.
2. En el dispositivo Android, abre el archivo descargado.
3. Si Android lo pide, permite **"Instalar apps de fuentes desconocidas"** para tu
   navegador o gestor de archivos.
4. Toca **Instalar**.

## Actualizar

Cada versión nueva se publica como un **Release** aquí. Para actualizar, baja el APK
de la versión nueva e instálalo encima del anterior (conserva tus datos y tu sesión).

> La app se conecta a `https://rp-mexico.com`. Necesitas una cuenta (negocio, guía o
> cliente) para entrar.
